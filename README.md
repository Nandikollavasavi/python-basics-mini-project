# Python Basics Mini Project
##10 days of python

## Overview

This project is a Python practice program created during my first 10 days of learning Python.

The project combines multiple Python concepts into a single application and demonstrates the use of variables, operators, conditions, loops, functions, strings, lists, tuples, dictionaries, and user input.

## Features

### 1. Student Information
- Accepts student details
- Displays name, age, branch, and college name

### 2. Marks Calculation
- Takes marks for three subjects
- Calculates total marks
- Calculates average marks

### 3. Grade Checker
- Assigns grades using conditional statements
- Displays A, B, C, D, or Fail

### 4. Even or Odd Checker
- Uses a function to determine whether a number is even or odd

### 5. Loop Practice
- Prints numbers from 1 to 10
- Prints even numbers from 0 to 20

### 6. String Operations
- Converts text to uppercase
- Converts text to lowercase
- Finds string length

### 7. List Operations
- Add items to a list
- Remove items from a list
- Find list length

### 8. Tuple Operations
- Access first and last elements

### 9. Dictionary Operations
- Add new key-value pairs
- Update existing values

### 10. Functions
- Demonstrates user-defined functions
- Displays student details using function parameters
